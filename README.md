# SalesAccounting
# Учёт реализации готовой продукции

## Описание
Программа для учёта продажи товаров (опт и розница).

## Основные данные
- Товары: название, оптовая цена, розничная цена
- Покупатели: название, адрес, телефон
- Продажи: покупатель, товар, количество, дата

## Технологии
C#, WPF, .NET Framework 4.8.1
